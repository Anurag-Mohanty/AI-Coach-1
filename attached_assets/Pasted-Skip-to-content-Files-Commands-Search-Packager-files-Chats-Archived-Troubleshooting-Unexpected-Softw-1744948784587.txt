🔧 Fix #1: Remove create_task() if it’s not async
If search_cached_content() is not an async function, don’t wrap it in asyncio.create_task():

python

Edit
cached_results = ContentCache().search_cached_content(context)

async def search_cached_content(self, context):

✅ But this only works if you’re doing async operations (e.g., file I/O or DB) inside that function.

Situation Fix
search_cached_content() is just returning a list ❌ Remove create_task() entirely
You want it to perform async I/O (e.g., file read, DB check) ✅ Make it async def and await it
You only want to cache in memory and it’s fast ✅ Keep it sync and call directly
